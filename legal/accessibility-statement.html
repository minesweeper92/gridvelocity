Accessibility Statement

Last updated 27 May 2026

This accessibility statement applies to gridvelocity.com, operated by Grid Velocity AOP. You can adjust display preferences at any time on our Accessibility page.

We want as many people as possible to be able to use this website. That means you should be able to:

  • Navigate the entire site using only a keyboard
  • Navigate using speech recognition software
  • Listen to most of the site using a screen reader (JAWS, NVDA, VoiceOver, TalkBack)
  • Increase text size up to 200% without content breaking or overlapping
  • Use the Quick Controls on our Accessibility page to adjust font size, contrast, motion, and link styling

Additional Accessibility Measures

We have taken the following steps to enhance accessibility across the site:

  • Built to WCAG 2.2 Level AA as a baseline standard
  • Semantic HTML with landmark roles, heading hierarchy, and descriptive alt text throughout
  • Full keyboard navigation with visible focus indicators on all interactive elements
  • We honour the prefers-reduced-motion system setting, and provide a manual Reduce Motion control on our Accessibility page for users whose OS doesn't expose that setting
  • Animations are always decorative, never the sole carrier of information
  • All colour combinations meet or exceed WCAG AA contrast ratios
  • Copy is written in plain, direct language throughout
  • Skip-to-content link on every page for keyboard and screen reader users

How Accessible Is This Website

WCAG 2.2 AA — Target

We target full WCAG 2.2 Level AA compliance across every page. Accessibility work is ongoing. We test pages manually and we act quickly when issues are reported.

We know some areas are still improving. These include:

  • Ensuring all third-party embedded content (maps, video players) is fully keyboard accessible
  • Continued testing across a wider range of assistive technologies and browsers

What We're Doing to Improve

Accessibility is not a launch-day checkbox. We treat it as ongoing work:

  • Every new page and component is reviewed against WCAG 2.2 AA before it ships
  • We test with keyboard-only navigation and screen readers during development
  • User feedback is reviewed and acted on — if you report something, it gets looked at

Feedback and Contact

If you encounter an accessibility barrier on this site — anything that doesn't work the way it should for you — we want to know. Tell us what page you were on, what browser and device you were using, and what assistive technology (if any) you were running.

Email us at hello@gridvelocity.com and put "Accessibility feedback" in the subject line. We'll get back to you within 5 working days.

If you need information from this site in a different format — large print, easy read, or audio — let us know and we'll do our best to accommodate you.

Technical Information

Grid Velocity is committed to making gridvelocity.com accessible in accordance with the Web Content Accessibility Guidelines (WCAG) 2.2, Level AA.

This site was built using semantic HTML5, CSS, and vanilla JavaScript. It uses GSAP for scroll-triggered animations and Lottie for vector animations — both are configured to respect reduced-motion preferences.
